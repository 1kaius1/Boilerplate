## Quick Start (Development)

### Go / Python

```bash
# 1. Clone
git clone https://github.com/OWNER/REPO.git
cd REPO

# 2. Configure environment
cp .env.example .env
# Edit .env and set required values

# 3. Set up the database
createdb app_name_dev          # PostgreSQL; skip for SQLite

# Go - run migrations
migrate -path migrations/ -database "${DATABASE_URL}" up

# Python - run migrations
alembic upgrade head

# 4. Build the frontend
npm install && npm run build

# 5. Start the server
go run ./cmd/app_name          # Go
uvicorn app_name.main:app --reload  # Python / FastAPI
flask run                           # Python / Flask

# Open: http://localhost:8080
```

### Laravel

```bash
# 1. Clone
git clone https://github.com/OWNER/REPO.git
cd REPO

# 2. Install PHP dependencies
composer install

# 3. Configure environment
cp .env.example .env
php artisan key:generate
# Edit .env - set DB_* and any other required values

# 4. Set up the database
# Create the database first if using PostgreSQL or MySQL:
# createdb app_name_dev  (PostgreSQL)
# mysql -e "CREATE DATABASE app_name_dev;"  (MySQL)

php artisan migrate
php artisan db:seed          # optional: seed development data

# 5. Install and compile frontend assets
npm install && npm run dev   # leave running for hot reload
# or for a one-time build:
npm install && npm run build

# 6. Start the development server
php artisan serve
# Open: http://localhost:8000

# If the app uses queues, start a worker in a separate terminal:
php artisan queue:work
```

## Configuration

Copy `.env.example` to `.env` and fill in the values. Never commit `.env`.

### Go / Python

```bash
# Required
DATABASE_URL=postgres://user:password@localhost:5432/app_name_dev
SECRET_KEY=change-this-to-a-long-random-string

# Optional
PORT=8080
ALLOWED_HOSTS=localhost,127.0.0.1
DEBUG=false
LOG_LEVEL=info
LOG_FORMAT=text
```

### Laravel

```bash
APP_NAME="App Name"
APP_ENV=local
APP_KEY=                     # generated by php artisan key:generate
APP_DEBUG=true               # false in production
APP_URL=http://localhost:8000

DB_CONNECTION=pgsql          # pgsql, mysql, or sqlite
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=app_name_dev
DB_USERNAME=your_user
DB_PASSWORD=your_password

CACHE_STORE=database         # database, redis, file
QUEUE_CONNECTION=database    # database, redis, sync
SESSION_DRIVER=database      # database, redis, file, cookie

MAIL_MAILER=log              # log (dev), smtp, mailgun, ses (prod)
```

## Database Migrations

### Go

```bash
migrate -path migrations/ -database "${DATABASE_URL}" up      # apply
migrate -path migrations/ -database "${DATABASE_URL}" version # status
migrate -path migrations/ -database "${DATABASE_URL}" down 1  # roll back one
```

### Python (Alembic)

```bash
alembic upgrade head     # apply
alembic current          # status
alembic downgrade -1     # roll back one
```

### Laravel

```bash
php artisan migrate               # apply pending migrations
php artisan migrate:status        # check migration state
php artisan migrate:rollback      # roll back the last batch
php artisan migrate:fresh         # drop all tables and re-run (dev only)
php artisan migrate:fresh --seed  # fresh migrate + seed (dev only)
```

> **Warning:** Never run `migrate:fresh` or `migrate:rollback` in production
> without explicit intent. Always back up the database before running migrations
> against production data.

## Deployment

### Laravel Production Checklist

Run these commands on every production deployment, in order:

```bash
composer install --no-dev --optimize-autoloader
php artisan config:cache
php artisan route:cache
php artisan view:cache
php artisan event:cache        # Laravel 11+
npm run build
php artisan migrate --force    # requires explicit confirmation to run in production
php artisan storage:link       # first deploy only, or if storage symlink is missing
```

To clear all caches after a bad deploy:

```bash
php artisan optimize:clear
```

### Docker

```bash
# Build image
docker build -t app_name:latest .

# Run container
docker run -d \
  --name app_name \
  -p 8080:8080 \
  --env-file .env.production \
  app_name:latest

# Laravel: run migrations after container starts
docker exec app_name php artisan migrate --force
```

### Docker Compose

```bash
docker compose up -d
# Laravel
docker compose exec app php artisan migrate --force
# Go / Python
docker compose exec app migrate -path migrations/ -database "${DATABASE_URL}" up

docker compose logs -f app
docker compose down
```
